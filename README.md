## Requirements
- [Node.js](https://nodejs.org/)
- [pnpm](https://pnpm.io/)

## Install
```sh
git clone https://github.com/TrAyZeN/asciiee.git
cd asciiee
pnpm install
```

## Usage
```sh
pnpm run start
```
