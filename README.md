## Requirements
- [Node.js](https://nodejs.org/)
- [yarn](https://yarnpkg.com/)

## Install
```
git clone https://github.com/TrAyZeN/asciiee.git
cd asciiee
yarn install
```

## Usage
```
yarn start
```
